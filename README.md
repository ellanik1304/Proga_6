# Proga_6

## Turtle Graphics Drawing

This repository contains a Python turtle graphics script that creates a drawing with geometric shapes and a grid of red dots.

### Running the Script

To run the turtle graphics script:

```bash
python3 turtle_drawing.py
```

### Requirements

- Python 3.x
- turtle module (included in Python standard library)

### What the Script Does

The script:
1. Sets up a large canvas (5000x5000)
2. Draws two sets of rectangular patterns
3. Creates a grid of 10,000 red dots (100x100 grid with spacing of 20 units)
