# Proga_6

## Turtle Graphics Drawing

This repository contains a Python turtle graphics script that creates a visual drawing with:
- Two rectangular shapes drawn with the turtle
- A grid of red dots covering the canvas

### Running the Script

To run the turtle graphics drawing:

```bash
python3 turtle_drawing.py
```

### Requirements

- Python 3.x
- tkinter (usually included with Python on systems with GUI support)

### What the Script Does

1. Sets up a large canvas (5000x5000 pixels)
2. Draws the first rectangular shape (6 iterations)
3. Moves to a new position
4. Draws the second rectangular shape (6 iterations)
5. Creates a grid of 10,000 red dots (100x100 grid) at 20-pixel intervals
